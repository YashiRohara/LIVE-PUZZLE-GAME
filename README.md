# 🧩 Live Puzzle Game

An interactive puzzle game built using Python, Pygame, and OpenCV that allows users to solve image-based puzzles with real-time gameplay.

---

## 🚀 Features
- 🎮 Interactive puzzle gameplay
- 🧩 Tile movement and highlighting
- 📸 Image-based puzzle generation
- 📷 Webcam integration (capture and play)
- ⏱️ Timer to track puzzle completion time

---

## 🛠️ Tech Stack
- Python
- Pygame
- OpenCV

---

## ▶️ How to Run
1. Install required libraries:
   pip install pygame opencv-python

2. Run the game:
   python main.py
